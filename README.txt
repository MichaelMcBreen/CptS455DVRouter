README

To run python 3 code, open up a corresponding number of terminals to routers.
Then run the program using your python 3 reserve word:
	router [-p] testdir routername

Trigger updates in a seperate terminal to print routing tables:
	printtables.py testdir r
	printtables.py testdir

Trigger updates in a seperate terminatl to change link cost: 
	changelink.py testdir r1 r2 c
